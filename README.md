# HeadHunter REST API

## Функционал проекта

- Разместить вакансию
- Разместить резюме
- Создать организацию/вступить в организацию
- Поиск по вакансиям(фильтры, сортировки)

- OpenApi автоматически генерирует документацию сваггерswagger
## Используемый стек технологий

- Java 16
- Spring Boot 2.6
- Swagger UI 
- MapStruct 
- OpenApi Codegen
- Spring Data JPA
- PostgreSQL
- Redis
